# Slovo

Офлайн-приложение для Android-телефона: чтение скачанных книг и прослушивание аудиокниг без регистрации, рекламы и дополнительного озвучивания текста.

## Возможности

- импорт TXT, FB2, EPUB и PDF через системный выбор файлов;
- импорт аудиокниг отдельными файлами или папкой с главами;
- библиотека с поиском и сохранением прогресса;
- жанровые и пользовательские полки, одна книга может находиться на нескольких полках;
- встроенный текстовый и PDF-ридер;
- постраничное чтение свайпами влево и вправо;
- регулировка яркости чтения свайпами вверх и вниз;
- аудиоплеер с перемоткой на 15 секунд назад и вперёд;
- тёмное оформление «Чернила и полночь»;
- вся информация хранится только на телефоне.

## Сборка

Нужны JDK 17, Android SDK 35 и Gradle 8.11.1.

```bash
gradle assembleDebug
```

APK появится в `app/build/outputs/apk/debug/app-debug.apk`. GitHub Actions также собирает APK автоматически после отправки изменений в `main`.
